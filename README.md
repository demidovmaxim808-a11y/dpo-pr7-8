# CarCostMaster - многостраничный сайт для учёта расходов на автомобиль

## Описание проекта
Многостраничный веб-сайт для учёта расходов на автомобиль, созданный в рамках практического занятия по фронтенд-разработке с использованием Bootstrap 5.

## Используемые технологии
- HTML5
- CSS3
- Bootstrap 5.3.0
- Bootstrap Icons 1.10.0
